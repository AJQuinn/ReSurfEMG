resurfemg.converter_functions module¶

This file contains functions to work with various EMG file types from various hardware/software combinations,and convert them down to an array that can be further processed with helper_functions or other modules. Additionally this file contains hash functions to track data

resurfemg.converter_functions.hash_it_up_right_all(origin_folder1, file_extension)¶

Hashing function to check files are not corrupted or to assure files are changed.

Parameters:

  • origin_folder1 – the string of the folder with files to hash

  • file_extension – file extension

Return df:

dataframe with hashes for what is in folder

Return type:

class:

pandas.DataFrame

resurfemg.converter_functions.poly5unpad(to_be_read)¶

This function converts a Poly5 read into an array without padding. Note there is a quirk in the python Poly5 interface that pads with zeros on the end.

Parameters:

to_be_read – filename of python read Poly5

Return df:

dataframe with hashes for what is in folder

Return type:

class:

~numpy.ndarray
